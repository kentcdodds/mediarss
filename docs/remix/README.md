# Remix package docs

MediaRSS runs on Remix v3 alpha APIs exposed through the unified `remix` package.

This directory mirrors the upstream package docs from the Remix monorepo at `remix@3.0.0-alpha.4`. Each package gets its own directory with the upstream `README.md` and any package-specific docs files for progressive disclosure.

## How to use these docs

- Start with the package `README.md`.
- Follow links into package-local docs when you need detail.
- Prefer these mirrored docs over ad-hoc web searches when changing Remix integrations in this repo.
